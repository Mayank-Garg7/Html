HTML Document Structure 

1. <!DOCTYPE html> 
    Declares the document type as HTML5. It ensures the browser interprets the page as HTML.
2. <html lang="en"> 
    Starts the HTML document and sets the language attribute to English ( en ).
3. <head> 
    Contains metadata about the HTML document, like its character set, title, and other settings.
4. <meta charset="UTF-8"> 
    Specifies the character encoding as UTF-8 to support various languages and symbols.
5. <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
    Ensures the webpage is responsive, adjusting the layout based on the device's screen size.
6. <title>Basic HTML Tags</title> 
    Sets the title of the webpage, displayed on the browser tab.

 Links 

1. <a href="../../../01-Introduction-to-HTML/" target="_blank">Learn more about HTML</a> 
    A hyperlink pointing to another webpage or resource.
    The  href  attribute specifies the link's URL.
    The  target="_blank"  attribute opens the link in a new tab.

 ---------------------------------------------------------------------------------------------
 

 Images 

1. <img src="../assets/images/nature.jpg" alt="HTML Example Image" width="400"> 
    Embeds an image into the webpage.
    src  specifies the file path of the image.
    alt  provides alternative text for accessibility.
    width  sets the image's width to 400 pixels.
