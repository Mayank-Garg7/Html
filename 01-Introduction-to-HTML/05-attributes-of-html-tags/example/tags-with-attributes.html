HTML Tags with Attributes

Attributes provide additional information about an element. Below are some examples:

Image Tag

The <img> tag displays an image. It has attributes like src, alt, width, and height.

Anchor Tag

The <a> tag creates a hyperlink. Common attributes include href and target.

List Elements

Lists are used to group related items:

Ordered List (OL)

The <ol> tag defines an ordered (numbered) list.

  1. First item
  2. Second item

Unordered List (UL)

The <ul> tag defines an unordered (bulleted) list.

Description List (DL)

The <dl> tag defines a description list with terms and their descriptions.
