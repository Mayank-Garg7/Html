HTML Text-Specific Tags

These tags modify the appearance or meaning of text:

Superscript: E = mc2 (superscript text).

Subscript: H2O (subscript text).

Small: This text is smaller than normal.

Big: This text is larger than normal.

Inserted Text: This text is newly added.

Deleted Text: This text is marked as deleted.

Blockquote: The blockquote tag is used for longer quotes:

"The greatest glory in living lies not in never falling, but in rising every time we fall." - Nelson Mandela

Q: The q tag is used for shorter inline quotes:

As they say, practice makes perfect.
