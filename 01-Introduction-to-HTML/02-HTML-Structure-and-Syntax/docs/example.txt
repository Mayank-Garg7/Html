HTML Structure: The Blueprint of a Webpage
--------------------------------------------------------------------------------------------------------------------------

What is HTML Structure?
HTML structure is like a blueprint for a webpage, similar to how a house needs a proper design with a foundation, walls, and rooms. It organizes elements (like headings, paragraphs, images, and lists) to display content logically and clearly on a web page.

--------------------------------------------------------------------------------------------------------------------------

Key Components of HTML Structure
Here are the main building blocks of an HTML page:

1. <html>
The root element of the HTML document. It contains all the other elements.
Tip: Always start and end your HTML with <html> and </html>.
Example:
<html>
  <!-- All content goes here -->
</html>
2. <head>
The information section of the page. It includes metadata, page title, styles, and scripts.
Example:
<head>
  <title>My First Page</title>
</head>
3. <body>
The visible section where you add headings, text, images, links, and more.
Example:
<body>
  <h1>Hello, World!</h1>
  <p>This is my first webpage.</p>
</body>

--------------------------------------------------------------------------------------------------------------------------

Commonly Used HTML Elements

Headings (<h1> to <h6>)

Create headings for titles and subheadings. <h1> is the largest, and <h6> is the smallest.
Example:
<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>
Paragraphs (<p>)
Add blocks of text for content.
Example:
<p>This is a paragraph of text explaining something.</p>
Lists
Unordered List (<ul>)
For bullet points.
Example:
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
Ordered List (<ol>)
For numbered points.
Example:
<ol>
  <li>Introduction</li>
  <li>Content</li>
  <li>Summary</li>
</ol>
Images (<img>)
Add visuals to your webpage.
Example:
<img src="photo.jpg" alt="A beautiful sunset">
Links (<a>)
Create clickable links to other pages or websites.
Example:
<a href="https://example.com">Visit Example</a>
Why is HTML Structure Important?
Readability: Makes your code easier to read, understand, and edit.
SEO (Search Engine Optimization): Helps search engines rank your page higher.
Accessibility: Supports screen readers and other assistive technologies.
Maintainability: A well-organized structure simplifies future updates.
Consistency: Improves user experience and keeps your site professional-looking.

Key Takeaways
HTML structure organizes your webpage content like a house blueprint.
Use essential elements (<html>, <head>, <body>, <h1>, <p>, <ul>, etc.) to build a well-structured page.
A good structure ensures better readability, SEO, accessibility, and ease of updates.
