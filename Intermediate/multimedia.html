HTML Multimedia with Attributes

HTML provides multimedia elements like images, audio, and video, with various attributes for customization and accessibility.

Image with Attributes

The <img> tag supports the following attributes:

  • src: Path to the image file (required).
  • alt: Alternative text for accessibility (recommended).
  • width, height: Set dimensions of the image.
  • title: Provides additional information on hover.

Audio with Attributes

The <audio> tag supports these attributes:

  • src: Path to the audio file (can also use <source>).
  • controls: Adds play, pause, and volume controls.
  • autoplay: Plays the audio automatically on load.
  • loop: Repeats the audio when it finishes.
  • muted: Starts the audio muted by default.

Video with Attributes

The <video> tag supports these attributes:

  • src: Path to the video file (can also use <source>).
  • controls: Adds play, pause, and volume controls.
  • autoplay: Plays the video automatically on load.
  • loop: Repeats the video when it finishes.
  • poster: Specifies an image to display before the video plays.
  • muted: Starts the video muted by default.

Embed with Attributes

The <embed> tag supports these attributes:

  • src: Path to the embedded content (e.g., PDF).
  • width, height: Set dimensions of the embedded content.
  • type: Specifies the type of embedded content (e.g., application/pdf).

Object with Attributes

The <object> tag supports these attributes:

  • data: Path to the object file (e.g., PDF, Flash).
  • type: Specifies the type of the embedded object (e.g., application/pdf).
  • width, height: Set dimensions of the object.
