HTML Formatting Tags

Use these tags to apply different styles to your text:

Bold: This text is bold.

Italic: This text is italicized.

Underline: This text is underlined.

Strikethrough: This text has a line through it.

Code/Monospace: This is monospace text, often used for code.

Highlighted: This text is highlighted.

These formatting tags help you present content in different styles.
